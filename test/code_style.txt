如果函数没有参数，则两个括号并在一起，如果有则，左右保留半个空格
for循环，如果元素为元素节点则命名为el，为普通节点（或夹杂其他类型节点）命名为node， 其他情况为item，序号则用i, j, k
函数中没有用到的参数名命名为_
字符串如果没有存在单双引号套嵌问题，统一使用双引号括起
不应该同时使用两种属性命名风格（colModel 和 table_body）
注释符号 `//` 后应该空一格
每个 function 之间多一空行，是的，除去注释外再多一空行；
适当加空格，比如 if 和后面的括号之间的空格、小括号和花括号之间的空格、冒号和 function 之间的空格等等；
风格上保持一致，你的代码里面有的地方 + 号和运算数之间有空格，有的则没有；
少用下划线开头的变量命名；
