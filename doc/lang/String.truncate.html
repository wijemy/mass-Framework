$.String.truncate( str, length?, truncation? )

描述：

将字符串截短为指定的长度（注意：这里的长度包含了后缀部分）， 并添加一个后缀（表示它仅只是一个摘要）。

参数：

  • length可选。Nunber。长度默认为 30。
  • truncation可选。String。默认后缀为 "..."。
例子 
$.require("lang", function() {
    $.log($.String.truncate("A random sentence whose length exceeds 30 characters."), true);
});
