object.forEach(fn, scope? )

描述：

对对象的键值使用fn进行迭代。

fn
必需。回调函数。
score
可选。要绑定的作用域。参数依次为value, key。
例子 
$.require("lang", function() {
    var obj = {
        first: 'Sunday',
        second: 'Monday',
        third: 'Tuesday'
    };
    $.Object.forEach(obj, function(v) {
        $.log(v, true);
    });
});
