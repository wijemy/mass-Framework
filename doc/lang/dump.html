$.dump( any )

描述：

此方法可以将javascript数据用很漂亮的缩进打印出来。

参数：

any
必需。任意类型。

返回值：

String。

例子 
require("lang,support", function($) {
    $.log($.dump("XXXX"), true);
    $.log($.dump($.support), true);
    var person = {
        name: "Jim Cowart",
        getName: function() {
            alert(1)
        },
        location: {
            city: {
                name: "Chattanooga",
                population: 167674
            },
            state: {
                name: "Tennessee",
                abbreviation: "TN",
                population: 6403000
            }
        },
        company: "appendTo"
    };
    $.log($.dump(person), true);

});
