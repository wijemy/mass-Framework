$.map(obj, callback)

描述：

将类数组与普通对象的元素或键值放进回调中，收集返回的结果。

回调函数的传参情况：

  1. 如果obj为类数组，那么依次为数组元素，当前索引值与类数组本身。
  2. 如果obj为对象，那么依次为值，键名与对象本身。

回调函数中的this始终指向第三个参数。

返回值：

一个新数组。

例子 
$.require("lang", function() {
    var ret = $.map({
        aa: "人生",
        bb: "工作",
        cc: "评议",
        dd: "祈祷"
    }, function(a, b) {
        return a
    });
    $.log($.dump(ret), true)

    ret = $.map(["Wii U", " PS Vita", "Xperia"], function(el) {
        return el + "XXX"
    });
    $.log($.dump(ret), true)
});
