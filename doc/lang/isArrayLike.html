$.isArrayLike

描述：

限定为Array, Arguments, NodeList与拥有非负整数的length属性的Object对象，视情况添加字符串

例子 
$.require("lang", function() {
    $.log($.isArrayLike([]), true);
    $.log($.isArrayLike(arguments), true);
    $.log($.isArrayLike(document.childNodes), true);
    $.log($.isArrayLike(document.getElementsByTagName("*")), true);
    $.log($.isArrayLike({
        0: 1,
        1: 2,
        2: 3,
        length: 3
    }), true);
});
