$.mix(target,source,[source2],[sourceN],[ride])

描述：

糅杂，为一个对象添加更多成员。具体做法，将第二个参数起的所有对象参数的键值都赋给第一个对象，如果最后一个参数为布尔值， 则决定是否用新值替代旧值，默认是替代。

参数：

target
必需。Object。 如果target为null,undefined,false,0,""等falsy值，则置换为一个新对象。
source
必需。Object。
source2
可选。Object。
sourceN
可选。Object。
ride
可选。Boolean。

返回值：

target

例子 
require("lang", function($) {
    var ret = $.mix({
        a: 1,
        b: 1,
        c: 1
    }, {
        aa: 2,
        bb: 2
    }, {
        cc: 3,
        dd: 3
    });
    ret = $.dump(ret);
    $.log(ret, true);
});
