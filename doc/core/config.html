$.cofig( opts )

描述：

用于配置框架的重要参数，比如调试时消息的显示级别，框架的别名，为模块进行别名。

参数：

opts
必需。Object。它的键值对将添加到$.config上！

默认地,它拥有如下默认值,当然你可以通过$.config({ level: 1})这样的方式修改它。

  • $.config.base, String， mass.js这个种子文件所在的目录,以/结束。
  • $.config.nick, String，框架的别名，默认为$。
  • $.config.alias, Object，用于放置框架的核心模块的别名。
  • $.config.level， Number，调试时消息的显示级别。

此外你可以另行设置其他项。

在框架的种子模块mass.js,它是这样设置这些值

    (function(scripts) {
        var cur = scripts[scripts.length - 1],
                url = (cur.hasAttribute ? cur.src : cur.getAttribute("src", 4)).replace(/[?#].*/, ""),
                kernel = $.config;
        basepath = kernel.base = url.slice(0, url.lastIndexOf("/") + 1);
        kernel.nick = cur.getAttribute("nick") || "$";
        kernel.alias = {};
        kernel.level = 9;
    })(DOC.getElementsByTagName("script"));;

使用:

 $.config.level = 3;//只打印最重要的出错信息
